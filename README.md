## docker setup

1. setup base image 

    ```bash
    cd docker-base
    bash setup.sh
    ```

2. setup app image

    ```bash
    cd ../docker-app
    bash setup.sh
    ```

3. start app

    ```bash
    bash start.sh
    ```
